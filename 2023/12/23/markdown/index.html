#+名称：为一级标题

eg：

一级标题

##+名称：为二级标题

eg:

二级标题

###+名称：为三级标题

eg：

三级标题

以此类推

markdown换行：在一行的末尾添加两个或多个空格，然后按回车键，即可创建一个换行

字体加粗：在单词或者短语的前后添加两个星号或者下划线 ，如果是加粗一部分，则在需要加粗部分的前后加入两个星号

eg:加粗

斜体：在单词或短语的前后各添加三个星号或者下划线，中间不要带空格

eg：斜体


//可以创建标签

模块引用，符号大于号可以表示

多个模块的块引用：为段落之间的空白行也加上模块引用符号（即大于号）

嵌套块引用，用两个模块引用符号（大于号）表示eg：>>

创建有序列表：

数字+英文.符号+内容

无序列表

在每个列表项前加-或*或+符号

要在列表中嵌套多个元素（即office中项目符号点的运用的样式）方法

在该元素前加四个空格或者一个制表符

代码块：

采用四个空格或一个制表符缩进。当他们放在列表中时，将他们缩进八个空格或者两个制表符

创建分割线方法

在单独一行上使用三个或多个星号（***）、破折号（—）或者（___)，并且不能包含其他内容

eg:

markdown链接语法（链接显示名，即链接文本放在中括号内，链接地址放小括号内

超链接markdown语法代码链接显示名,即链接显示文本

eg:

这是一个链接 [个人博客]（1xy123.github.io)

如果想对链接添加标题，则在链接地址后先加个空格，再将要加的标题放在两个双引号之间

输入email，将其放在一对尖括号之间

带格式化链接

eg:强调链接：在链接前后加入*，若将链接表示为代码，请在方括号中加入反括号

添加图片

输入![]和小括号

#markdown转义字符，在字符前面加反斜杠\

特殊字符自动转义

html标签使用，将其名字放在<>中

markdown表格

使用三个或多个连字符（—)创建每列的标题，并使用管道|分割每列，可以选择表的任一端增加管道符号

eg:

Syntax Description
|Header Title
paragraph text
